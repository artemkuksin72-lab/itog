# Expense Tracker

**Автор:** Куксин Артем Алексеевич

## Описание
Программа для контроля личных финансов. Позволяет записывать расходы, классифицировать их по категориям и анализировать затраты за всё время или по конкретным категориям.

## Функции
- Добавление расхода (Сумма, Категория, Дата).
- Валидация данных: сумма > 0, проверка формата даты.
- Фильтрация по категориям.
- Автоматический подсчет итоговой суммы (в т.ч. при фильтрации).
- Сохранение данных в JSON-файл.

## Как использовать
1. Запустите скрипт: `python main.py`.
2. Введите сумму расхода.
3. Выберите категорию из списка.
4. Укажите дату (по умолчанию стоит сегодняшняя).
5. Нажмите "Добавить расход".
6. Используйте выпадающий список фильтра для просмотра трат по конкретной категории.
